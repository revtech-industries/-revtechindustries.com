# Artemis AI Pitch Deck

A professional, interactive pitch deck for Artemis AI - the world's first intelligent mental health companion.

## Features

- **7 Comprehensive Slides** covering all key investor points
- **Interactive Navigation** with keyboard, mouse, and touch support
- **Responsive Design** that works on desktop, tablet, and mobile
- **Professional Styling** with modern UI/UX design
- **Print-Ready** format for PDF export
- **Smooth Animations** and transitions

## Slide Content

1. **The Problem** - Mental health crisis and current solution gaps
2. **The Solution** - Artemis AI's unique approach and features
3. **Product Demo** - Key features and technical specifications
4. **Market Opportunity** - Market size, growth, and target segments
5. **Business Model** - Revenue streams and financial projections
6. **Traction & Progress** - Current milestones and achievements
7. **What We Need** - Funding ask and partnership opportunities

## Navigation

- **Mouse**: Use Previous/Next buttons or click slide indicators
- **Keyboard**: Arrow keys (←/→), Space bar, Home/End
- **Touch**: Swipe left/right on mobile devices

## Usage

### Web Presentation
1. Open `index.html` in a web browser
2. Use navigation controls to move between slides
3. Present in fullscreen mode for best experience

### PDF Export
1. Open in browser and press Ctrl+P (Cmd+P on Mac)
2. Select "Save as PDF" as destination
3. Choose "More settings" and select "Background graphics"
4. Save the PDF for offline presentations

### Customization
- Edit `index.html` to modify slide content
- Update `pitch.css` to change styling and colors
- Modify `pitch.js` to add custom functionality
